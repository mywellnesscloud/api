Appointments

Appointments are meetings involving a user and a facility staff member. They can either be scheduled by the end user or by a staff member.

Set Appointment

Schedule a meeting with a facility staff member.

Url:
https://services.mywellness.com/api/[version]/Facility/[FacilityId]/SetAppointment
HttpMethod:
POST

Parameters

token: string required
user access token
name: string required
meeting title
userId: string required
user id
assignedTo: string required
the staff the user wants to meet with
dueDate: date required
the meeting date
startHourTime: int required
the meeting start hour as integer value
startMinuteTime: int required
the meeting start minute as integer value
endHourTime: int required
the meeting end hour as integer value
endMinuteTime: int required
the meeting end minute as integer value
