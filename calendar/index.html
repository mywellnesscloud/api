Classes Calendar

Users are mywellness.com registered users that use technogym ecosystem tracking their activities, biometrics, manage theirs training program, challenges and training plan. Resources available on a User are: Profile, Aspiration, ActivitySummary.

Search Calendar Events

Retrieve the details for the event having the given ID.

Url:
https://services.mywellness.com/api/[version]/Facility/[FacilityId]/CalendarEvents
HttpMethod:
POST

Parameters

date: int required
the base date for search query
timeScope: string required
set the time range for the query (see table below for available ranges)
staffMember: string
if specified filters the query by staff id
eventTypeId: string
if specified filters the query by event type id
roleId: string
if specified filters the query by role id
eventType: string
if specified filters the query by event type (see table below for defined types)
room: string
if specified filters the query by room name
dateStart: int required when timeScope is set to Custom
start date for the search query
dateEnd: int required when timeScope is set to Custom
end date for the search query
token: string required
user access token
Event types
id: 1 name: Class
Calendar Time Scopes
id: 0 name: Custom
id: 1 name: Day
id: 2 name: Week
id: 3 name: Month 
            { 
              "token" :""

              "timeScope": "Week",

              "roleId": "7827cb2f-f366-4747-9724-1a879be842c4",

Event Details

Retrieve the details for the event having the given ID.

Url:
https://services.mywellness.com/api/[version]/CalendarEvent/[EventId]/Details
HttpMethod:
POST

Parameters

token: string required
user access token 
            { 
              "token" :""
            }

Response

name: string
event name
room: string
event's room
calendarEventType: string
a string reporting the event type (see the event types table)
calendarEventTypeId: string
unique id for the event type
assignedTo: string
id of the staff the event is assigned to
assignedToDescription: string
name of the staff the event is assigned to
facilityId: string
unique id of the facility
pictureUrl: string
url to the event picture
recurrence: object
object describing the event's recurrence
