Staff

Users are mywellness.com registered users that use technogym ecosystem tracking their activities, biometrics, manage theirs training program, challenges and training plan. Resources available on a User are: Profile, Aspiration, ActivitySummary.

Staff Availability

Query the staff availabilities for the current facility.

Url:
https://services.mywellness.com/api/[version]/Facility/[facilityId]/BrowseStaffAvailability
HttpMethod:
POST

Parameters

token: string required
user access token
startDate: int required
start date for the current query
endDate: int required
end date for the current query
staffId: string
filter results by specific staff
roleId: string
filter results by specific role
